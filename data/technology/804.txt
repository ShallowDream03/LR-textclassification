 要说起目前上市面上最受欢迎的小尺寸平板电脑，那肯定非iPad mini莫属。无论是时尚的外观、轻薄小巧的机身，还是iOS系统的独特性，都吸引了众多消费者前来购买。虽然iPad mini并不是第一款问世的小尺寸平板电脑，但它的出现无疑让全世界刮起了一阵“小尺寸风暴”。 虽然iPad mini是一款不错的产品，但这种一家独大的境况对于整个行业的发展并不是什么好事，许多消费者还是希望涌现出更多更好的小尺寸平板电脑来与之竞争。于是 这次三星电子再次为我们带来了一款让人惊艳的产品，那就是拥有8英寸轻薄机身的Galaxy Note 8.0通话平板电脑。便携性不相上下 Galaxy Note 8.0拥有一块8英寸大小的屏幕，与iPad mini的7.9英寸的屏幕看上去基本一致。而在厚度方面，Galaxy Note 8.0的机身厚度为7.95mm,而iPad mini为7.2mm，不到一毫米的差距基本可以忽略不计。如果从便携性方面来说，两款产品基本是不相上下的。外观方面各有千秋 iPad mini如此受追捧的一大原因便是它拥有了一个十分时尚的外观，白色的前面板加上航空级铝合金材质的后壳的确极具诱惑力。而三星Galaxy Note 8.0作为一款新时代的平板电脑，在外观方面自然而然也采用了十分时尚的设计。除了独特的椭圆鹅卵石型外观设计能为我们提供够舒适的持握感之外，通体纯白 色的机身也无透露着的时尚气息。 总的来说Galaxy Note 8.0在外观设计方面还是透露着浓浓的“三星风格”，与之前三星发布的Galaxy系列有着许多相似之处。如果非要评比出它跟iPad mini在外观方面谁更胜一筹，那只能根据消费者的喜好来评判了。毕竟仁者见仁，每个人的审美都是不一样的。 那么抛开外观、便携性这些方面不谈，Galaxy Note 8.0相比iPad mini又有什么优势呢？经过笔者的一系列使用体验，个人认为Galaxy Note 8.0在实用性方面是比iPad mini强太多了。接下来，笔者便罗列出三点，来向大家证明Galaxy Note 8.0这款产品到底有多实用。 多窗口任务运行 通过在Android系统的上的深度改造，三星在原有系统单一窗口显示的基础上开发出了能够更好的利用现有大尺寸屏幕的多窗口视图。这个功能就好像是我们 平时经常使用的Windows系统中的横向平铺窗口视图，只不过是窗口的数量被限定在了两个罢了。那么在这种模式下整个屏幕的空间被两个应用充分的利用， 这对于同时需要对两个完整应用进行操作的用户来说是很有必要的。而对于iPad mini来说，是完全没有办法实现这个功能的。多窗口运行功能 这个功能绝对是个十分实用的功能，想象一下当用户在查看电子邮件中的附件时，突然发现有一些比较重要内容需要记下来，或者是在日历中设置一个项目提醒等。 此时我们就可以通过屏幕侧边的多窗口托盘调出备忘录或者是日历应用到屏幕的一侧，在不影响邮件查看的情况下执行其它的操作。可以调节两个任务界面的大小 那么将传统电脑桌面上的多窗口任务操作带到了平板电脑上，三星采取的这种做法大大提高了对平板电脑大屏幕的使用效率，可能这也是很多用户之前想到却没有厂 商做到的事情。那么作为原生Android系统所不具备的功能，这个多窗口视图也证明了三星的二次研发实力，也确实从交互操作上带给了用户不同于iPad mini等同类产品的体验。 S Pen触控笔让操作更方便 S Pen触控笔也是Galaxy Note 8.0的一大特色，除了了用于手写以及触控操作之外，它还可以做很多的事情。比如我们可以用它进行截图，用户需要在按住笔杆上的功能键后使用笔尖在 NOTE 8.0的屏幕上按照自己的需要滑出一个闭合的图形，当闭合图形画完之后这部分内容就就会被截取下来，并且系统会自动打开图片编辑器，使用者可以为它添加一 些批注，或者是直接拖拽到S Note应用中作为素材使用。Note 8.0附带的S pen触控笔各色功能让操作更加方便 除此之外，我们还可以借助S Pen触笔上的侧向功能键，来对设备设置一些手势操作。根据用户滑出的不同的符号，系统会激活相应的功能，这无疑对我们平时的使用提供了许多的帮助。当然 对于笔者来说，觉得对我们帮助最大的莫过于“浮窗预览”这个功能了。当我们把S Pen触笔的笔尖悬在某个屏幕中的元素上时，就会浮出一个窗口显示与这个元素相关的信息。总的来说，S Pen的设计为我们生活中带来了许多方便，依靠这根笔我们可以让Galaxy Note 8.0做更多的事情。 通话功能 笔者最后一个要介绍的就是Galaxy Note 8.0所拥有的通话功能，个人认为这也是它相比iPad mini最大的一个优势。8英寸的机身尺寸虽然比大部分通讯手机要大了太多太多，但我们依然可以一只手就可以把它拿在耳边进行接打电话。当然如果你觉得这 依然不是很方便，我们还可以通过外接蓝牙耳机、线控耳机来进行通话功能。单手便可接打电话 也就是说，Galaxy Note 8.0是融合了平板与手机的一款产品，强大的硬件配置以及大尺寸的屏幕也能秒杀目前市面上的主流手机。如果你也认为出门带一部iPad mini同时还需带一部手机太麻烦的话，那么入手Galaxy Note 8.0绝对是你的最佳选择。
